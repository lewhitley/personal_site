My Skills

Spotlight utilizes a Ruby on Rails backend, a PostgreSQL database, and ReactJS with a Redux architectural framework on the frontend. I leveraged ActiveRecord to pre-fetch track data contained in other tables and minimize N+1 queries, allowing for a faster and more seamless UX. To make annotation entry seamless, I used jQuery to locate clicks on the screen, informing where the form should render to allow the user to still view the lyrics.

Christmas Lights was implemented with Vanilla JavaScript for overall structure and game logic, HTML5 Canvas for DOM manipulation and rendering, and Webpack to bundle and serve up the various scripts. The lit paths are found with a Depth-First Search.

Hoo is a single-page Vue.js application with Vuex frontend structure built on Ruby on Rails and PostgreSQL. Components are built with the single-file structure, keeping HTML, CSS, and JavaScript for the component in one place. The event search feature makes AJAX requests to the backend to take advantage of ActiveRecord's strengths and efficiently filter by the parameters.
